# Exploration Document Template

Write to `docs/research/NNN-<slug>.md` (next sequential number in the directory).

```markdown
# <Title>

**Date:** YYYY-MM-DD
**Status:** Open | Decided | Superseded
**Decision:** [If decided, link to ADR. Otherwise "Pending"]

## Question

[The framed question from Phase 1]

## Current State

[Summary from Phase 2 — what exists, constraints, relevant code paths]

## Landscape

[Summary from Phase 3 — prior art, libraries, best practices]

## Strategies Evaluated

### Strategy A: <Name>
[Description, strengths, weaknesses, effort, risk]

### Strategy B: <Name>
...

### Strategy C: <Name>
...

## Debate Summary

[Key arguments from the quorum. What was decisive? What nearly won?]

## Recommendation

[Pick a winner. State it clearly. Include conditions under which you'd reconsider.]

## Open Questions

[What still needs answering before implementation? What should the ADR address?]

## Sources

[Numbered list with URLs]
```
